# Jeremy Hayes Arts site

This folder contains the first static build for the Jeremy Hayes Arts website.

## Render setup

Create a new Static Site in the same Render workspace as Draw Studio.

Repository: `charlottearthub/cah-draw-studio`

Root Directory: `jeremyhayesarts`

Build Command: leave blank

Publish Directory: `.`

Auto Deploy: On Commit

The site will use `index.html`, `styles.css`, and optional future files inside this folder.
